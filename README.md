# KnightLauncher (iOS)

KnightLauncher is a launcher that allows you to play **Spiral Knights** on your iOS device. It is a fork of [Amethyst](https://github.com/AngelAuraMC/Amethyst-iOS) and [PojavLauncher](https://github.com/PojavLauncherTeam/PojavLauncher).

## Introduction

* KnightLauncher is a dedicated launcher for the Java-based MMO **Spiral Knights**.
* It handles the downloading, installation, and execution of the game on iOS.
* It utilizes OpenJDK to run the game natively.
* Supports JIT for better performance (requires TrollStore or debugger).

## Requirements

* iOS 14.0 or later.
* [Amethyst](https://github.com/AngelAuraMC/Amethyst-iOS): The immediate parent fork.
* [PojavLauncher](https://github.com/PojavLauncherTeam/PojavLauncher): The base of this project.
* [Spiral Knights](https://www.spiralknights.com/): The game itself (owned by Grey Havens).
* [Boardwalk](https://github.com/zhuowei/Boardwalk): The original project.
* [Caciocavallo](https://github.com/PojavLauncherTeam/caciocavallo), [GL4ES](https://github.com/ptitSeb/gl4es), [MoltenVK](https://github.com/KhronosGroup/MoltenVK): Key libraries for iOS support.
